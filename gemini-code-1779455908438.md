# UX Flow: '선 무지성 입력, 후 초고속 분류'

## 1. 데이터 입력 (Input Flow)
- 사용자가 앱을 켜면 무조건 **[Inbox]** 화면이 가장 먼저 보인다.
- 카테고리 선택 없이 텍스트(제목+내용)만 입력하고 Enter를 치면 Inbox에 데이터가 적재된다.

## 2. 데이터 분류 (Routing Flow)
- Inbox에 쌓인 데이터 카드에는 [태그 추가] 버튼이 노출된다.
- 사용자가 태그 버튼을 눌러 29개 매장 중 하나(예: '#나의 일기')를 선택하면, 해당 데이터는 Inbox 화면에서 즉시 사라지고(Fade-out) 해당 매장(카테고리)으로 이동된다.

## 3. 조회 및 수정 (View Flow)
- 좌측 사이드바에서 특정 매장을 클릭하면, Master DB에서 해당 태그를 가진 데이터만 필터링되어 메인 화면에 뿌려진다.
