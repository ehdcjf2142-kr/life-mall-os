# Project: Life Mall OS (개인용 올인원 라이프 관제 앱)

## 1. 프로젝트 목적
사용자의 과거, 현재, 미래의 모든 데이터를 단 하나의 애플리케이션에서 통제하고 관리하는 '개인용 라이프 OS' 구축.

## 2. 핵심 아키텍처 철학 (Core Philosophy)
- **Single Database, Multi-Views:** 모든 데이터는 단 하나의 마스터 데이터베이스(Master DB)에 저장된다. 폴더 구조를 절대 사용하지 않으며, 오직 '태그(Tag)' 필터링을 통해서만 29개의 세부 카테고리 화면으로 렌더링한다.
- **Zero Friction (마찰 제로):** 사용자가 데이터를 입력할 때 카테고리를 고민하지 않도록, 최상단에 'Inbox(수신함)'를 배치하여 무조건 빠른 텍스트 입력이 가능해야 한다.

## 3. 주요 메타포 (쇼핑몰 5대 레이어)
앱의 구조는 5개 층으로 이루어진 '프리미엄 쇼핑몰'의 개념을 차용한다.
- 4F: 총괄 경영관 (미래/전략)
- 3F: 감성 문화관 (지식/영감)
- 2F: 금융 & 편집숍 (자산/장비)
- 1F: 라이프 마켓 (실시간 실행)
- B1: 기계실 & 창고 (인프라/유지보수)
