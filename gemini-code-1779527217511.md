# Project: Life Mall OS - Immersive Visual Overhaul (Frontend Only)

## 1. 개요 및 목적 (Context)
현재 구현된 UI(첨부 이미지 0, 1)는 기능적으로는 작동하지만, 기획 의도인 '프리미엄 인생 쇼핑몰'의 몰입감이 전혀 느껴지지 않습니다. 
이에 **백엔드(Gemini API 연결, Database 구축) 작업은 완전히 배제하고**, 오직 **프론트엔드(UI 컴포넌트 렌더링,UX 동선)만** 시각적으로 대폭 Refactoring하여 쇼핑몰 콘셉트를 완성하고자 합니다.

## 2. 개발 제한사항 (Strict Constraints)
- **NO API Integration:** Gemini API 호출 로직을 절대 구현하지 마십시오.
- **NO Database/Backend:** 실제 데이터베이스를 연결하거나 백엔드 로직을 만들지 마십시오.
- **Mock Data Only:** 모든 층, 매장, 메모 목록은 프론트엔드 코드 내에 하드코딩된 **가짜 데이터(Mock Data)**를 사용하여 렌더링하십시오.

## 3. 핵심 수정 사상 (Core Revision Philosophy)
- **Dashboard to Department Store:** 기존의 지루한 텍스트 리스트 기반 대시보드 형태를 버리고, **비주얼 맵(Visual Map)** 방식의 레이아웃을 도입한다.
- **Visual Discovery Path:** 루트 화면(조감도) -> 층 화면(도면) -> 상점 화면(인벤토리)으로 이어지는 시각적인 동선을 구축한다.

## 4. 상세 수정 스펙 (Detailed Specs)

### A. 루트 화면: [인생 쇼핑몰 조감도 뷰] (New Root Component)
- 사용자가 앱을 켜면, 지금처럼 Inbox가 먼저 나오는 것이 아니라 메인 영역에 **[Life Mall OS 백화점의 세련된 조감도/외부 전경 일러스트]**가 렌더링되어야 합니다.
- raster 이미지를 쓰는 것이 아니라, 코드로 비주얼 구조를 렌더링하십시오 (SVG, Canvas 라이브러리 활용 등).
- **UX:** 백화점 그림의 각 층 영역에 마우스를 올리면 하이라이트되고, 클릭하면 해당 층의 내부(Floor View)로 React Router를 통해 시각적으로 이동합니다.
- (기존 좌측 사이드바는 접어두거나, 디자인을 더 미니멀하게 변경하여 비주얼 영역을 넓힙니다.)

### B. 층별 화면: [매장 도면 뷰] (New Floor View Component)
- 특정 층(예: 2F)을 클릭하면, 메인 영역이 **[해당 층의 평면도/도면]** 스타일의 비주얼로 바뀝니다.
- 도면 위에는 층별 카테고리(매장들)가 작은 '상점 아이콘'이나 '미니 샵프론트(Shopfront)' 형태로 비주얼하게 배치됩니다.
- **UX:** 원하는 상점 비주얼을 클릭하면 해당 상점 내부(기존 데이터 리스트 뷰)로 진입합니다.

### C. Inbox 화면: [물류 터미널 UI + AI 추천 Mockup] (Immersive Inbox UI)
- **UI:** 기존의 단순한 입력 폼을 **[택배 상자가 가득 찬 물류 터미널 / 신상품 언박싱 존]** 콘셉트로 디자인을 대폭 변경합니다. 입력창 디자인을 '박스' 형태로 만듭니다.
- **AI 추천 Mockup:** (실제 API 연결은 하지 않지만) Inbox 입력 필드 하단에 **"[상점 추천] 메모 내용을 분석 중입니다..."**라는 placeholder 텍스트를 미리 디자인해 둡니다. 나중에 API를 붙였을 때 렌더링될 위치를 잡아두는 것입니다.

### D. 상점 화면: [콘셉트 인벤토리 뷰] (Refactored Shop View)
- (image_1.png 같은 빈 화면 대신) 특정 상점 진입 시, 메인 영역의 데이터 리스트 테마를 해당 상점의 콘셉트(예: 중앙은행 = 금고 형태, 심야 라운지 = 아늑한 서재 형태)로 Wrap하여 렌더링하십시오. 흑백 모노톤 기반의 세련된 디자인을 유지합니다.

## 5. 결과물 예상 시나리오 (Expected Output Flow)
1. **앱 진입:** 멋진 백화점 조감도 그림이 보임.
2. **1F 클릭:** 특가 할인과 광장이 있는 도면 비주얼로 바뀜.
3. **나의 프로젝트(팝업스토어) 클릭:** 쇼핑몰 가판대 같은 테마로 wrap된 프로젝트 가짜 데이터(Mock data) 목록이 보여짐.
4. **Inbox 클릭:** 물류 창고 같은 멋진 화면이 나옴. AI 추천placeholder 텍스트가 예쁘게 박혀있음.
