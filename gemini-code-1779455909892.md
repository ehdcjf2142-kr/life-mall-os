# Database Schema & Category Mapping

## 1. Master DB Schema (단일 테이블 구조)
- id: string (UUID)
- title: string (기록 제목)
- content: text (본문 내용, 마크다운 지원)
- floor: string (B1, 1F, 2F, 3F, 4F)
- tag_category: string (29개 매장 중 택 1)
- created_at: timestamp
- is_inbox: boolean (true면 Inbox 표시, false면 태그 기반 렌더링)
