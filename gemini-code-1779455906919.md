# UI Layout & Design System

## 1. 전반적인 테마
- **디자인 스타일:** 불필요한 장식이 없는 모노톤(흑백) 기반의 극단적인 미니멀리즘. 데이터(텍스트) 자체가 디자인이 되도록 여백(Padding/Margin)을 넉넉하게 사용.
- **다크 모드 지원:** 기본 테마는 다크 모드 기반으로 설정 (배경: #121212, 텍스트: #E0E0E0).

## 2. 레이아웃 구조 (2-Column Layout)
- **[Left Sidebar] 네비게이션 패널 (Width: 250px 고정)**
  - 최상단: '📥 무지성 Inbox' 버튼 (가장 크게 강조)
  - 중간 영역: 4F부터 B1까지 5개의 층(대분류)이 아코디언 메뉴로 배치. 각 층을 클릭하면 하위 29개 카테고리(매장)가 펼쳐짐.
- **[Right Main] 메인 대시보드 (Width: Flex)**
  - 상단: 선택된 매장(카테고리)의 제목과 1줄 설명 출력.
  - 중앙: 데이터 리스트 출력 (보드 뷰, 리스트 뷰, 갤러리 뷰 등 카테고리 성격에 맞춰 렌더링).
  - 우측 하단: 플로팅 액션 버튼(FAB)으로 언제든 빠른 메모 추가 가능.
